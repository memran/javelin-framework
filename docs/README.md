## Reading Order

If you are new to Javelin:

1. Start with [Core](modules/core.md).
2. Read [Starter](modules/starter.md) for the default boot path.
3. Use the module pages for the feature you want to add.
4. Use [Console](modules/console.md) for the workspace toolchain and scaffolding commands.
5. Each module page explains the main functions, when to call them, and what they return.
